周报（二）250515重生之我是CS完全新手

发表于 2025-05-15 更新于 2025-07-21 分类于 记事
本文字数： 901 阅读时长 ≈ 2 分钟

就是啊，怎么中间缺了一周呢？

清单

  1. CS50 3/10
  2. 高数基础 2/24
  3. C语言程序设计 9/12

1

我寻思就算通过了初试，那又该怎样提升自己在复试中的竞争力呢？这时我发现了CS自学指南和CS自学社区并感到醍醐灌顶，顿时觉得自己本来打算的学习路径过于莽撞和平庸，打比方的话就是想通过一本字典就学会一门外语，困难又低效。

上周的想法是这周手里的考研资料齐全了，可以全心全意投入到学习中去了，结果发现了新大陆，然后自己也更愿意学。另外英语单词任务改成了每天25+75，反正这样也能在今年12月考6级之前结束掉，刚好凑个整嘛。之所以看cs50不直接CS61A的原因是数学基础还没打好，反正cs50的作业对我这个笨蛋来说已经够够的了。

这周30号考科目一来着，最好暑假直接就开练科目二，不然学生都放假教练也不好约。下周打算花更多功夫到数学上，因为心心念念的UCB CS61全家桶有前置知识要求。

2

目前准备的打基础的学习资料如下，数字序号下是打算同时并行学习的内容：

    • 武忠祥数高数基础 + 1987-2008整理真题
    • cs50x + 作业 + 《C语言程序设计》
    • cs61a + 《深入理解计算机系统》 + 作业
    • 武忠祥高数基础 + 1987-2008整理真题
    • cs61b + 作业
    • 王道数据结构网课 + 讲义 + 习题 + 《严蔚敏数据结构（C语言版）》
    • 姜晓千线代基础 + 1987-2008整理真题
    • cs61c + 作业
    • 王道计组网课 + 讲义 + 习题 + 《计算机组成与系统结构》
    • 姜晓千概率基础 + 1987-2008整理真题
    • NJU OS网课 + 作业 + 《操作系统导论》
    • 王道os网课 + 讲义 + 习题
    • 660
    • USTC Computer Networking + 《计算机网络 自顶向下方法》
    • 王道计网网课 + 讲义 + 习题

……还有每天背单词。整理材料的时候是很热心啦，但就算我还没进入大三，我真的来得及学完这些吗？从现在五月中旬，到明年九月报名这期间？我想想，我还打算考英语六级、参加蓝桥杯、暑假考驾照、应付学校课程、作业和考试、准备毕业设计……然后数学强化课程和习题是需要的吧？408历年真题是需要仔细研究的吧？英语作文的历年真题和作文模版都是需要准备的吧？政治再怎么敷衍，总归要了解了解的吧？唉，好忙。
